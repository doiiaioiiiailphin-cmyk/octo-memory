# 🎮 吃豆人 (Pac-Man)

一个经典街机游戏《吃豆人》的网页版复刻，使用纯 HTML5 Canvas + JavaScript 实现。

![Pac-Man Screenshot](https://via.placeholder.com/560x620/000000/FFFF00?text=Pac-Man+Game)

## 🎯 游戏特色

- **完整游戏逻辑** - 包含吃豆人、幽灵、能量豆等经典元素
- **智能幽灵AI** - 幽灵会追逐玩家，吃能量豆后会逃跑
- **多关卡系统** - 吃完所有豆子后自动进入下一关
- **分数系统** - 吃豆子、能量豆、吃幽灵都有分数
- **生命系统** - 3条生命，被幽灵抓住会损失生命
- **暂停功能** - 随时暂停继续游戏
- **流畅动画** - 吃豆人张嘴动画、幽灵移动动画

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| `↑` / `W` | 向上移动 |
| `↓` / `S` | 向下移动 |
| `←` / `A` | 向左移动 |
| `→` / `D` | 向右移动 |
| `空格` | 暂停/继续游戏 |

## 🎨 游戏元素

### 得分规则
- **小豆子**: 10 分
- **能量豆**: 50 分
- **吃掉幽灵**: 200 分

### 游戏角色
- **🟡 吃豆人** - 玩家控制的角色
- **🔴 Blinky (红色幽灵)** - 直接追逐型
- **🌸 Pinky (粉色幽灵)** - 伏击型
- **🔵 Inky (青色幽灵)** - 随机型
- **🟠 Clyde (橙色幽灵)** - 随机型

### 特殊道具
- **能量豆** - 大白点，吃到后幽灵变蓝，可以反过来吃掉幽灵（持续7秒）

## 🚀 如何运行

### 方法1: 直接打开
双击 `pacman.html` 文件，在浏览器中打开即可游玩

### 方法2: 本地服务器
```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node.js
npx serve
```
然后访问 `http://localhost:8080/pacman.html`

## 🛠️ 技术栈

- **HTML5 Canvas** - 游戏渲染
- **纯 JavaScript** - 游戏逻辑（无框架依赖）
- **CSS3** - 界面样式

## 🎯 游戏目标

1. 控制吃豆人吃掉迷宫中的所有豆子
2. 躲避四个幽灵的追击
3. 吃能量豆后可以反击幽灵
4. 完成一关后进入下一关，难度递增
